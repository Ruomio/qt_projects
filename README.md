## widget_demo
### 安装教程

1.  Qt，OpenCV，Cmake，Make工具
2.  进入到项目CmakeLists目录，新建build文件夹，进入到build文件夹 使用`cmake ..`命令，会在build文件夹生成可执行文件

### 使用说明

1.  图库太大就没上传，需要把自己的数据集放在 widget_demo/trains/images中
2.  需要把 face_train.yml 删除或重命名， 因为程序中判断存在这个文件就不训练数据集
